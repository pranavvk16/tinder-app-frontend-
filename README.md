# Tinder App Frontend

React Native/Expo Tinder-like dating app.

## Features
- Splash screen
- Main screen (likely swiping/matching)
- Liked People screen
- Account screen
- API integration via src/services/api.js (backend at http://127.0.0.1:8000/api)

## Setup
1. Clone repo
2. `npm install`
3. Copy .env.example to .env and set API_BASE_URL
4. `npx expo start`

## Structure
- `src/pages/`: Screens (SplashScreen.js, MainScreen.js, etc.)
- `src/services/api.js`: Axios API client
- `assets/`: Icons, splash
